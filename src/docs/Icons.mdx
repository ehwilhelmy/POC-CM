# Icons

We use **@mui/icons-material** (Material Design icons) throughout the system.

## Installation

```bash
npm install @mui/icons-material @mui/material @emotion/styled @emotion/react
```

## Usage

```tsx
import SearchIcon from '@mui/icons-material/Search';
import CloseIcon from '@mui/icons-material/Close';

// Default 24px
<SearchIcon />

// Sized via fontSize prop
<SearchIcon fontSize="small" />  // 20px
<SearchIcon fontSize="large" />  // 35px

// Custom size via sx or style
<SearchIcon style={{ fontSize: 16 }} />
```

## Icon Size Tokens

Map MUI's sizing to our design tokens:

| Token | Value | MUI equivalent |
|-------|-------|----------------|
| `--icon-sm` | 16px | `style={{ fontSize: 16 }}` |
| `--icon-md` | 20px | `fontSize="small"` |
| `--icon-lg` | 24px | default |
| `--icon-xl` | 32px | `style={{ fontSize: 32 }}` |
| `--icon-xxl` | 40px | `style={{ fontSize: 40 }}` |
