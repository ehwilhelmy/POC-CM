# Shadows & Elevation

Three shadow levels for layering and depth. Defined in `src/theme/tokens.css`.

---

## When to Use

| Level | Token | Use case |
|-------|-------|----------|
| Subtle | `--shadow-sm` | Cards, inputs at rest |
| Default | `--shadow-md` | Hover states, dropdowns |
| Elevated | `--shadow-lg` | Modals, side panels, floating elements |

### Usage

```css
.cm-card {
  box-shadow: var(--shadow-sm);
}

.cm-card:hover {
  box-shadow: var(--shadow-md);
}

.cm-sidepanel {
  box-shadow: var(--shadow-lg);
}
```
