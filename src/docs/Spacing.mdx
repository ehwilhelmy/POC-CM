# Spacing

A consistent spacing scale used for padding, margin, and gaps. Defined in `src/theme/tokens.css`.

## Scale

<div style={{ display: 'flex', flexDirection: 'column', gap: 12 }}>
  {[
    { token: '--space-xs', value: '4px' },
    { token: '--space-sm', value: '8px' },
    { token: '--space-md', value: '12px' },
    { token: '--space-lg', value: '16px' },
    { token: '--space-xl', value: '24px' },
    { token: '--space-xxl', value: '32px' },
    { token: '--space-xxxl', value: '40px' },
  ].map(s => (
    <div key={s.token} style={{ display: 'flex', alignItems: 'center', gap: 16 }}>
      <code style={{ width: 120, fontSize: 12, flexShrink: 0 }}>{s.token}</code>
      <div style={{ width: 48, fontSize: 12, color: '#808080', flexShrink: 0, textAlign: 'right' }}>{s.value}</div>
      <div style={{ width: parseInt(s.value), height: 24, background: '#380f8c', borderRadius: 2, flexShrink: 0 }} />
    </div>
  ))}
</div>

## Border Radius

<div style={{ display: 'flex', flexWrap: 'wrap', gap: 24, marginTop: 8 }}>
  {[
    { token: '--radius-xs', value: '2px' },
    { token: '--radius-sm', value: '4px' },
    { token: '--radius-md', value: '6px' },
    { token: '--radius-lg', value: '8px' },
    { token: '--radius-xl', value: '12px' },
    { token: '--radius-xxl', value: '16px' },
  ].map(r => (
    <div key={r.token} style={{ textAlign: 'center' }}>
      <div style={{
        width: 64,
        height: 64,
        background: '#efe8f7',
        border: '2px solid #380f8c',
        borderRadius: r.value,
      }} />
      <div style={{ fontSize: 11, marginTop: 4, color: '#505050' }}>{r.value}</div>
      <code style={{ fontSize: 10, color: '#808080' }}>{r.token}</code>
    </div>
  ))}
</div>

### Usage

```css
.cm-card {
  padding: var(--space-lg);
  margin-bottom: var(--space-xl);
  border-radius: var(--radius-lg);
  gap: var(--space-sm);
}
```
